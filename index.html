DateTimePicker

Klick auf den Trigger öffnet das Popup. Tag / Monat / Jahr / Stunde / Minute via Tastatur, Mausrad oder Pfeiltasten anpassen, mit Enter bestätigen.

Source: github.com/Ranudar/datetime-picker · Single-file demo, kein Build-Schritt am Client. React via esm.sh (importmap); Komponente vorab via esbuild gebündelt und inline. Direkt aus dem Browser über file:// öffenbar (Internet-Verbindung für CDN-React erforderlich).
