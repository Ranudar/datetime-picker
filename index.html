DateTimePicker

Klick auf den Trigger öffnet das Popup. Tag / Monat / Jahr / Stunde / Minute via Tastatur, Mausrad oder Pfeiltasten anpassen, mit Enter bestätigen.

Source: github.com/Ranudar/datetime-picker · Single self-contained file — React, ReactDOM und die Komponente sind alle inline. Funktioniert offline, direkt per file:// aus dem Browser geöffnet.
